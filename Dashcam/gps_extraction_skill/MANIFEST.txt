QUICK START:
------------

1. Install prerequisites (see INSTALL.md)
2. Run: ./run.sh /path/to/crop/directory
3. View results in: output/gps_extraction_*.xlsx

EXPECTED RESULTS:
-----------------

Processing time: 40-60 minutes for ~28,000 images
Success rate: 85-90%
GPS extracted: ~24,500 coordinates
Output: Excel file with lat/lon coordinates

CRITICAL PARAMETERS:
--------------------

Crop width: 0.70 (70%) minimum - DO NOT REDUCE
Crop height: 0.22 (22%) recommended
Workers: 4 (GPU parallel workers)
OCR: PaddleOCR with GPU acceleration

DEPENDENCIES:
-------------

- Python 3.11
- PaddleOCR >= 2.7.0
- pandas >= 2.0.0
- NVIDIA GPU with CUDA support
- 20GB+ free disk space
